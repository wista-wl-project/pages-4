# Code of Conduct

## 行動規範について

WISTA プロジェクトは、  
**非商用かつ自由な動画視聴環境を守ること**を目的とした  
コミュニティ主導のプロジェクトです。

本行動規範は、  
開発者・利用者・貢献者すべてが安心して参加できる環境を維持するためのものです。  
本プロジェクトに関わるすべての方は、以下の内容を尊重してください。

---

## 私たちの約束

WISTA コミュニティは、以下を約束します。

- 立場や経験の違いを尊重します
- 建設的かつ誠実なコミュニケーションを行います
- 批判は人ではなく内容に向けます
- プロジェクトの理念（非商用・自由利用）を尊重します
- 法的条件（ライセンス）を遵守します

---

## 期待される行動

以下の行動を推奨します。

- 丁寧で礼儀正しい言葉遣い
- 建設的なフィードバック
- 初心者への配慮とサポート
- 意見の違いを冷静に議論する姿勢
- Issue / Pull Request のルールを守ること

---

## 許容されない行動

以下の行為は禁止します。

- 攻撃的、侮辱的、差別的な発言や行動
- ハラスメント（言語的・視覚的・技術的を含む）
- スパム、荒らし行為
- 脅迫、なりすまし、嫌がらせ
- 他者のプライバシー侵害
- プロジェクトの理念を無視した行為
- **無断での商用利用、再配布、ライセンス違反行為**
- Issue や PR を用いた営利目的の宣伝行為

---

## 適用範囲

本行動規範は、以下のすべてに適用されます。

- GitHub 上の Issue / Pull Request / Discussions
- プロジェクトに関連するコミュニケーション全般
- 公開・非公開を問わず、WISTA に関連する場

---

## 違反への対応

行動規範に違反する行為が確認された場合、  
Wista Project は以下の対応を行う場合があります。

- 注意・警告
- Issue / PR のクローズ
- コメントや投稿の削除
- 一時的または恒久的なアクセス制限（ブロック）
- 必要に応じた追加措置

対応は、状況と影響を考慮し、誠実かつ段階的に行います。

---

## 報告方法

行動規範違反と思われる行為を見かけた場合は、  
以下のいずれかの方法でご連絡ください。

- GitHub Issue（非公開が望ましい場合はその旨を明記してください）
- メール: wista-project@outlook.jp

可能な範囲で、  
発生日時・内容・該当URL等を記載してください。

---

## 最後に

WISTA は、  
**誰かを排除するためのプロジェクトではありません。**

自由で非商用な環境を守るために、  
互いを尊重し、誠実に関わることを大切にしています。

この行動規範に同意いただける方のみ、  
WISTA コミュニティへの参加をお願いします。

ありがとうございます。
