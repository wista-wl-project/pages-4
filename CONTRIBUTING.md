# Contributing to WISTA

WISTA プロジェクトへの関心と貢献、ありがとうございます。  
本ドキュメントは、WISTA に対する貢献方法とルールを定めたものです。

本プロジェクトは  
**非商用・強コピーレフト・事前通知義務ライセンス（WNCCL-PN）**  
のもとで運営されています。  
貢献を行う前に、必ず本内容をご確認ください。

---

## 基本方針

WISTA は、以下の理念を重視しています。

- 非商用かつ自由な動画視聴環境の維持
- クローズド化・商用独占の防止
- 透明性のある開発と運営
- 利用者・開発者双方にとって誠実なプロジェクト運営

これらの方針に同意いただける場合のみ、貢献をお願いします。

---

## 貢献の種類

以下のような貢献を歓迎します。

- バグ報告
- 機能提案・改善案
- UI / UX の改善
- ドキュメント修正
- 翻訳（日本語・英語）
- コードの修正・追加

---

## Issue の作成について

### バグ報告

Issue を作成する際は、可能な限り以下の情報を含めてください。

- 発生している問題の詳細
- 再現手順
- 利用環境（ブラウザ / OS 等）
- スクリーンショット（可能であれば）

---

### 機能提案・改善案

- 既存の Issue に同様の提案がないか確認してください
- なぜその機能が必要かを明確にしてください
- プロジェクトの方針（非商用・自由利用）に沿っているかをご確認ください

---

## Pull Request について

### 事前相談のお願い（重要）

**コードの変更を伴う Pull Request を作成する前に、  
必ず Issue を立てて事前に内容を共有してください。**

これは以下の理由によるものです。

- ライセンス方針との整合性確認
- 実装方針のすれ違い防止
- 不要な作業の発生を防ぐため

---

### Pull Request のルール

- 変更内容が分かる明確な説明を記載してください
- 関連する Issue がある場合は必ずリンクしてください
- 大規模な変更は、複数の PR に分割してください
- フォーマット・コーディングスタイルは既存コードに合わせてください

---

## フォーク・改変・再配布について（重要）

WISTA は改変・Fork・再配布を許可していますが、  
**以下の条件を必ず守ってください。**

- 商用利用は禁止されています
- 同一ライセンス（WNCCL-PN v1.0）での公開が必須です
- **実施前に Wista Project への事前通知が必須**です
- 派生物であることを明確に表示してください
- ソースコードを公開してください

### 事前通知先

- GitHub Issue: https://github.com/wista-project
- メール: wista-project@outlook.jp

---

## 商用利用について

WISTA は非商用利用のみ許可されています。  
商用利用・業務利用・収益化を伴う利用については、  
**必ず事前に個別ライセンスの相談を行ってください。**

商用利用の相談は、  
GitHub の「商用利用ライセンスの問い合わせ」Issue テンプレート  
またはメールにて受け付けています。

---

## 行動規範について

他の貢献者や利用者に対し、  
敬意をもって接してください。

以下の行為は禁止します。

- 攻撃的・差別的な発言
- スパム行為
- プロジェクトの理念を無視した行為
- 無断での商用利用や再配布

問題行為が確認された場合、  
Issue / PR のクローズ、ブロック等の対応を行う場合があります。

---

## 最後に

WISTA は、  
**自由で非商用な動画視聴環境を守るためのプロジェクト**です。

この理念に共感し、  
一緒に育ててくれる方の貢献を歓迎します。

ありがとうございます。
