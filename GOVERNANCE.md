# Governance（運営方針）

本ドキュメントは、  
WISTA プロジェクトにおける **運営体制・意思決定・責任範囲** を明確にするものです。

WISTA は、  
**非商用・自由・持続可能**であることを最優先とした  
コミュニティ主導のオープンソースプロジェクトです。

---

## プロジェクトの位置づけ

- WISTA は **個人主導プロジェクト** です
- 特定の企業・団体・スポンサーに依存しません
- 商用化を目的としたプロジェクトではありません
- 利用者・貢献者の協力によって成り立っています

---

## 運営主体

### Wista Project

以下を総称して **Wista Project** と呼びます。

- リポジトリ管理者
- メンテナー
- 中核開発者

Wista Project は、  
プロジェクト全体の方針決定と最終責任を負います。

---

## 意思決定プロセス

### 最終決定権

以下の事項については、  
**Wista Project が最終決定権**を持ちます。

- ライセンスおよび利用条件
- 商用利用の可否判断
- 破壊的変更（MAJOR バージョン）
- セキュリティ対応
- 行動規範違反への対応

---

### 通常の意思決定

- Issue / Pull Request / Discussion を通じて議論します
- 合理的な合意形成を目指します
- 必要に応じてメンテナーが判断します

---

## コントリビューターの役割

### コントリビューターとは

以下を行う方をコントリビューターと呼びます。

- Issue の報告
- Pull Request の提出
- ドキュメント改善
- 翻訳・検証・テスト

コントリビューターは、  
プロジェクトに対する **提案権** を持ちますが、  
**決定権は持ちません**。

---

## メンテナーについて

- メンテナーは Wista Project により選出されます
- 継続的かつ誠実な貢献が基準となります
- メンテナーはレビュー・方針提案を行います
- 権限は必要最小限に付与されます

---

## 商用利用に関する判断

- 商用利用は原則禁止です
- 例外的な許諾は、個別に判断します
- 許諾条件は非公開とする場合があります
- 判断理由の公開義務はありません

---

## ライセンスとガバナンス

- 本プロジェクトは独自ライセンスを採用しています
- ライセンス条件はガバナンスの中核です
- ライセンス違反への対応は運営判断で行います

---

## 透明性について

WISTA は、  
可能な限り透明な運営を目指します。

ただし、以下については公開されない場合があります。

- セキュリティに関わる内容
- 個人情報を含む事項
- 商用利用交渉の詳細
- トラブル対応の内部判断

---

## 紛争・トラブルへの対応

- 感情的対立よりも事実を重視します
- 公開の場での過度な議論は制限される場合があります
- 必要に応じて Issue / PR をクローズします
- 最終的な対応は Wista Project が決定します

---

## ガバナンスの変更

- 本ドキュメントは変更される場合があります
- 重要な変更は事前に告知します
- 変更後も過去の利用条件を遡及的に無効化するものではありません

---

## 最後に

WISTA は、  
**誰かの支配や利益のためのプロジェクトではありません。**

自由で非商用な環境を守るため、  
明確な運営方針を定めています。

この方針に理解・同意いただける方の参加を歓迎します。
