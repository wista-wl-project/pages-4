# FAQ（よくある質問・誤解対策）

本ドキュメントは、  
WISTA プロジェクトに関して **特に誤解されやすい点** を中心にまとめた FAQ です。

利用・改変・再配布・商用利用を検討している方は、  
必ず最後までご確認ください。

---

## Q1. WISTA は自由に使っていいのですか？

**はい、非商用利用に限り自由に利用できます。**

WISTA は  
**Wista 非商用コピーレフト・事前通知義務ライセンス（WNCCL-PN）**  
のもとで公開されています。

ただし、以下の条件があります。

- 商用利用は禁止
- 改変・再配布・Fork には事前通知が必要
- 同一ライセンスでの公開（ライセンス継承）が必須

---

## Q2. 商用利用とは何を指しますか？

以下に該当する場合、**商用利用**とみなされます。

- 広告を掲載する
- 課金・サブスクリプションを行う
- サービス・製品の一部として提供する
- 企業・団体の業務用途として利用する
- 利益・対価が発生する可能性がある利用

判断に迷う場合は、  
**必ず事前に商用問い合わせ Issue を作成してください。**

---

## Q3. 商用利用したい場合はどうすればいいですか？

README に記載されている  
**「商用利用したい場合はこちら」** から、  
商用問い合わせ用 Issue を作成してください。

👉 Website: https://wista.f5.si

Issue をもって **自動的に許可されるわけではありません**。  
内容を確認のうえ、Wista Project より対応方針を連絡します。

---

## Q4. 個人で自分用に使うだけなら通知は不要ですか？

**改変しない利用であれば、通知は不要です。**

ただし、以下の場合は **事前通知が必須** です。

- ソースコードを改変する
- Fork して公開する
- 再配布する
- デプロイして第三者がアクセスできる状態にする

---

## Q5. デプロイ（公開）はしていいのですか？

**はい、非商用であれば可能です。**

Render / GitHub Pages / その他ホスティングを含め、  
デプロイ自体は禁止していません。

ただし、

- 非商用であること
- 事前通知を行うこと
- ライセンス表記を保持すること

が条件です。

---

## Q6. Fork は自由にしていいのですか？

Fork 自体は可能ですが、  
**公開・配布・改変を伴う場合は事前通知が必須**です。

また、

- 同一ライセンスでの公開
- ライセンス表記の保持
- 作者・プロジェクトの明示

を必ず守ってください。

---

## Q7. ライセンスを変更して公開できますか？

**できません。**

WISTA は **コピーレフト** ライセンスです。  
派生物・改変版は必ず  
**WNCCL-PN と同一条件のライセンス**で公開してください。

---

## Q8. このプロジェクトは違法ですか？

WISTA は  
**クライアント側で公開情報を取得・再生する技術的実装**を提供する  
オープンソースプロジェクトです。

- 違法アップロードを推奨しません
- 著作権侵害行為を助長しません
- 利用者の責任において使用されます

各国の法律・利用規約を遵守した利用を行ってください。

---

## Q9. YouTube の代替を名乗って問題ありませんか？

「代替」という表現は  
**機能的な代替を示すものであり、公式な関係を示すものではありません。**

WISTA は Google / YouTube とは無関係の  
独立したオープンソースプロジェクトです。

---

## Q10. API や外部サービスの責任は誰が負いますか？

WISTA は複数の外部 API・サービスを利用しますが、

- 各 API の利用条件は利用者自身が確認する必要があります
- Wista Project は外部サービスの仕様変更・停止について責任を負いません

---

## Q11. 無断商用利用・ライセンス違反を見つけたら？

以下のいずれかでご連絡ください。

- GitHub Issue
- メール: wista-project@outlook.jp

状況に応じて、  
警告・対応・必要な措置を行います。

---

## Q12. なぜここまで条件が厳しいのですか？

WISTA は、

- 自由で非商用な利用環境を守るため
- 悪質な転売・商用搾取を防ぐため
- 開発者・利用者双方を守るため

あえて **明確で強い条件** を設けています。

理解・尊重していただける方の参加を歓迎します。

---

- GitHub Issue
- 商用問い合わせテンプレ
- メール: wista-project@outlook.jp

誠実な相談は歓迎します。
